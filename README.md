A dynamic, high-performance personal portfolio site built on Salesforce Experience Cloud (LWR). Engineered with a revolutionary "Zero-Hardcode" architecture for seamless content management—no code deploys needed!

✨ Key Features
Scalable "Zero-Hardcode" Design: Content fully managed via Custom Metadata—update projects, skills, or bios without deployments.

High-Performance Backend: Custom Apex service layer handles complex data models with guest-user security and real-time optimization.

Advanced Reactive UI: Lightning Web Components (LWC) with multi-dimensional filtering, GitHub API integration, and fully responsive layouts.

Enterprise-Grade Delivery: Production-ready LWR site architecture emphasizing maintainability, security, and scalability.

Clone the repo: git clone <your-repo-url>

Push to Salesforce: sf project deploy start

Configure Custom Metadata with your portfolio data.

Activate LWR site in Experience Builder.

Customize GitHub API token in Named Credentials.
